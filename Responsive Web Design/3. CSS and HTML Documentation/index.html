Introduction to HTML and CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are fundamental technologies for structuring and styling web pages.

  • HTML provides the foundational elements of a webpage, defining its content and structure.
  • CSS controls the presentation of the webpage, allowing customization of layouts, colors, fonts, and responsiveness.

Together, they create visually appealing and functional web experiences that work across different devices and screen sizes.

Basic HTML Structure

An HTML document consists of elements organized hierarchically to define a webpageâ€™s content.

Key Elements:

  • <!DOCTYPE html>: Specifies the document type and version of HTML (usually HTML5).
  • <html>: The root element encompassing all content.
  • <head>: Contains metadata, including character encoding, viewport settings, and external CSS or scripts.
  • <title>: Defines the text displayed on the browser tab.
  • <body>: Holds all visible content such as text, images, and links.
  • Semantic elements like <header>, <footer>, and <article> provide meaningful structure to improve readability and accessibility.
CSS Basics

CSS (Cascading Style Sheets) is used to control the appearance of HTML elements, ensuring consistency in design and layout.

Selectors and Properties

  • Selectors define which HTML elements the CSS rules apply to.
  • Properties specify the styling rules, such as font size, color, and margin.

Common Selectors

  • Element Selector (element****): Targets all instances of a specific HTML element.
  • Class Selector(.classname****): Targets elements with a specific class.
  • ID Selector (#id****): Targets a unique element by its ID.
  • Attribute Selector ([attribute]****): Targets a unique element by its ID.
  • Pseudo-classes (:hover****, :focus): Apply styles when an element is in a specific state.
Layout Techniques

Flexbox

Flexbox is a one-dimensional layout system that helps in distributing space efficiently among items.

  • display: flex; enables a flexible contianer.
  • justify-content controls horizontal alignment.
  • align-items manages vertical alignment.

Grid Layout

CSS Grid provides a two-dimensional layout model for web design.

  • display: grid; activates a grid container.
  • grid-template-columns and grid-template-rows define the structure of the grid.
  • gap specifies spacing between grid items.
Responsive Design

Responsive design ensures that a webpage adapts to different screen sizes and devices.

Media Queries

Media queries apply different styles based on screen size conditions.

  • @media (max-width: 600px) {} targets screens smaller than 600 pixels.
  • @media (min-width: 768px) and (max-width: 1024px) {} applies style for tablets.
Forms in HTML

Forms allow users to input data and interact with websites.

Key Elements:

  • <form>: Defines the form
  • <label>: Provides a description for input fields.
  • <input>: Accepts user input (text, email, password, etc.).
  • <textarea>: Allows multiline text input.
  • <button>: Triggers form submission.
  • <select>: Creates dropdown menus.
Styling Forms with CSS

Styling enhances form usability and appearance.

Important CSS Properties

  • padding creates spacing inside input fields.
  • border defines input field boundaries.
  • border-radius adds rounded corners.
  • background-color modifies button backgrounds.
  • cursor: pointer; changes the cursor to indicate clickable elements.
Typography in CSS

typography_in_css affects readability and aesthetics.

Key Properties

  • font-family defines the typeface.
  • font-size controls text size.
  • font-weight adjusts thickness (bold, normal, light).
  • line-height determines vertical spacing between lines.
  • letter-spacing modifies spacing between cahracters.
Colors and Backgrounds

CSS provides control over text, background, and gradient colors.

Essential Properties:

  • color changes text-color.
  • background-color sets a solid background.
  • background-image applies images as backgrounds.
  • opacity controls transparency.
  • gradient effects create smooth transitions between colors.
Box Model

The box model explains how elements are sized and spaced.

Components:

  • Content: The area containing text or images.
  • Padding: The space inside the border.
  • Border: The element's outer edge.
  • Margin: The space outside the border seperating elements.
